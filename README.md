# restaurant_list
# 餐廳清單
餐廳清單，提供餐廳資訊及搜尋。

## 功能
餐廳清單瀏覽
查看任一餐廳個別詳細餐廳資訊
搜尋餐廳名稱或類別

### 使用環境

需安裝 node.js 及 npm 

### 安裝步驟

1. 開啟終端機後, 將本專案複製一份至本機電腦

2. 開啟專案的資料夾

3. 安裝 npm 套件

4. 安裝 nodemon 套件

5. 執行程式

6. 當終端機出現"Express is listening on localhost:3000"字樣
表示伺服器與資料庫成功連結並成功啟動

## 開發工具

1. Node.js
2. Express 4.17.2
3. Express-Handlebars 5.3.3
4. Bootstrap 5.0
5. Font-awesome
